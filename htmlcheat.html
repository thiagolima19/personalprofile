Cheat Sheet

HTML

<!DOCTYPE html> It informs the browsing about the document type.
<html> </html> Creates a HTML document.
<head> </head> Sets off the title and other informations, such language.
<meta> Defines metadata about HTML document.
<title> </title> Sets off the title of the webpage.
<link> Defines the relationship between the current document and an external resource.
<script> </script> Used to embed a clients-side scripty such JavaScript.
<body> </body> Sets off the visible part of the webpage.
<a> </a> It defines a hyperlink. href=" " is where you put the link address.
<h1-h6> </h1-h6> Creates a headings. h1 is the most important and h6 is the least.
<p> </p> It creats a new paragraph.
<nav> </nav> Defines a set of navigation links.
<br> </br> Inserts a line break.
<div> </div> it defines a section and is used as a container to be styled by CSS.
<span> </span> Is used to format inline content with CSS.
<pre> </pre> Defines preformatted text.
<ul> </ul> Creates an unordered list.
<ol> </ol> Creates an ordered list.
<li> </li> Is used to create each item in the list.
<form> </form> Difines a form.
<table> </table> Creates a table.
<tr> </tr> Defines the table header.
<td> </td> Sets each cell in a row.
<button> </button> Creates a clickable button.
<!-- --> Creates a comment that is not showed on the webpage.
<img src="" alt=""> Is used to embed an image in an HTML page.
<hr> Is used to create a horizontal rule in the webpage.
