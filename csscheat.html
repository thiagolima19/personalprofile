Cheat Sheet

CSS

#ID selector: it is a unique identification that is attributed to an HTML element. e.g. #id{}
.Class selector: it is an identification that is attributed to an HTML element. e.g. .class{}
* Universal selector: selects all the elements in the HTML page. e.g. *{}
Tag name: the name of the HTML tag is used for styling. e.g. p{}
pseudo selector: is used to define a special state of an element. e.g. :root{}
align-itens: specifies the alignment inside of flex container
animation: shorthand property to all animation property.
animation-delay: delay for start the animation.
animation-direction: direction for the animation.
animation-name: defines the name for the @keyframes animation.
aspect-ratio: specify the aspect ratio for an element.
background: shorthand property to all background property.
background-attachement: sets if the image scrolls together or will be fixed.
background-color: sets a color to the background.
background-image: sets a image background.
background-repeat: sets if it will be repeat.
background-size: sets the size background image.
border: shorthand property to all borders.
border-color: sets the border color.
border-radius: shorthand for all four border radius.
border-style: sets the border style.
color: sets the text color.
columns: shorthand for column-width and column-count.
cursor: sets the mouse cursor when is pointing over an element.
display: sets how a certain HTML element should be displayed.
flex: shorthand for flex-grow, flex-shrink, flax-basis.
font: shorthand for font properties.
grid: A shorthand for the grid property.
height: sets the height of an element.
justify-content: align items inside a flexible container.
justify-items: align items on a grid container.
@keyframes: specifies the animation code.
margin: shorthand for all margins properties.
overflow: sets what happens to an element when its content goes out of limits.
padding: shorthand for all padding properties.
scroll-behaviour: specifies whether goes straight to the section or in a smooth way.
text-align: specifies the horizontal text alignment.
text-decoration: specifies the text decoration.
transform: applies a 2D or 3D transformation to an element.
transition: shorthand for all transition properties.
translate: specifies the position of an element.
width: specifies the width of an element.
z-index sets the stack order of a positioned element.
